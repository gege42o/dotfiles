# My dotfiles

Oh my zsh + .zshrc configs and install script for new system

It contains:
- Oh-My-Zsh install
- zsh install
- pipes screensaver
- fastfetch
- nvim install
- btop
- net-tools
- omz plugins
- docker+compose

Copy this to your terminal and run it:

```bash
git clone https://github.com/gege42o/dotfiles.git
cd dotfiles
chmod +x install.sh
./install.sh
```
